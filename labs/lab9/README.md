## –ó–∞–¥–∞—á–∞ 1 (–ë–∏–±–ª–∏–æ—Ç–µ–∫–∞)

[–ò—Å—Ç–æ—á–Ω–∏–∫](https://github.com/Alvant/AdvancedPython/tree/master2022/labs/lab06#%D0%B7%D0%B0%D0%B4%D0%B0%D1%87%D0%B0-1-%D0%B1%D0%B8%D0%B1%D0%BB%D0%B8%D0%BE%D1%82%D0%B5%D0%BA%D0%B0).

–í –±–∏–±–ª–∏–æ—Ç–µ–∫–∞—Ö —Ö—Ä–∞–Ω—è—Ç—Å—è –∫–Ω–∏–≥–∏.
–£ –∫–∞–∂–¥–æ–π –∫–Ω–∏–≥–∏ –µ—Å—Ç—å –Ω–∞–∑–≤–∞–Ω–∏–µ, –∞ —Ç–∞–∫–∂–µ –∞–≤—Ç–æ—Ä, –∏ –¥–∞—Ç–∞ –≤—ã–ø—É—Å–∫–∞.
–£ –∫–∞–∂–¥–æ–π –±–∏–±–ª–∏–æ—Ç–µ–∫–∏ —Ç–æ–∂–µ –µ—Å—Ç—å –Ω–∞–∑–≤–∞–Ω–∏–µ.
–í –±–∏–±–ª–∏–æ—Ç–µ–∫–µ –º–æ–∂–µ—Ç –±—ã—Ç—å –±–æ–ª—å—à–µ –æ–¥–Ω–æ–≥–æ —ç–∫–∑–µ–º–ø–ª—è—Ä–∞ –∫–Ω–∏–≥–∏ —Å —Ç–µ–º–∏ –∂–µ –Ω–∞–∑–≤–∞–Ω–∏–µ–º –∏ –∞–≤—Ç–æ—Ä–æ–º.
–ù–∞–¥–æ –æ–ø—Ä–µ–¥–µ–ª–∏—Ç—å –∫–ª–∞—Å—Å—ã –¥–ª—è *–ö–Ω–∏–≥–∏* –∏ *–ë–∏–±–ª–∏–æ—Ç–µ–∫–∏*.

–ü—Ä–∏ —ç—Ç–æ–º –¥–æ–ª–∂–Ω–∞ –±—ã—Ç—å –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç—å –¥–æ–±–∞–≤–∏—Ç—å –∫–Ω–∏–≥—É –≤ –±–∏–±–ª–∏–æ—Ç–µ–∫—É —Å –ø–æ–º–æ—â—å—é –æ–ø–µ—Ä–∞—Ü–∏–∏ `+`.
–ü—Ä–∏ —Å–ª–æ–∂–µ–Ω–∏–∏ –±–∏–±–ª–∏–æ—Ç–µ–∫–∏ –∏ –∫–Ω–∏–≥–∏ –ø–æ–ª—É—á–∞–µ—Ç—Å—è –Ω–æ–≤–∞—è –±–∏–±–ª–∏–æ—Ç–µ–∫–∞ —Å —Ç–µ–º –∂–µ –Ω–∞–∑–≤–∞–Ω–∏–µ–º, —á—Ç–æ —É –±–∏–±–ª–∏–æ—Ç–µ–∫–∏-—Å–ª–∞–≥–∞–µ–º–æ–≥–æ, –Ω–æ —É –∫–æ—Ç–æ—Ä–æ–π —Å—Ä–µ–¥–∏ –∫–Ω–∏–≥ –µ—Å—Ç—å –µ—â—ë –æ–¥–Ω–∞ ‚Äî –∫–Ω–∏–≥–∞-—Å–ª–∞–≥–∞–µ–º–æ–µ.
–ù–∞–ø—Ä–∏–º–µ—Ä:
```python
biblio = Library(name="–ë–∏–±–ª–∏–æ—Ç–µ–∫–∞ –ú–§–¢–ò")
book = Book(
    name="–¢–µ–æ—Ä–µ—Ç–∏—á–µ—Å–∫–∞—è —Ñ–∏–∑–∏–∫–∞. –¢–æ–º 1. –ú–µ—Ö–∞–Ω–∏–∫–∞",
    author="–õ–∞–Ω–¥–∞—É –õ.–î., –õ–∏—Ñ—à–∏—Ü –ï.–ú.",
    year=2004
)

biblio = biblio + book
# –¢–µ–ø–µ—Ä—å –≤ –±–∏–±–ª–∏–æ—Ç–µ–∫–µ biblio –µ—Å—Ç—å –æ–¥–Ω–∞ –∫–Ω–∏–≥–∞
```

–î–æ–ª–∂–Ω–∞ –±—ã—Ç—å –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç—å —Å—Ä–∞–≤–Ω–∏—Ç—å –¥–≤–µ –±–∏–±–ª–∏–æ—Ç–µ–∫–∏: —Å—á–∏—Ç–∞–µ—Ç—Å—è, —á—Ç–æ –æ–¥–Ω–∞ –±–∏–±–ª–∏–æ—Ç–µ–∫–∞ "–±–æ–ª—å—à–µ" –¥—Ä—É–≥–æ–π, –µ—Å–ª–∏ –≤ –Ω–µ–π –±–æ–ª—å—à–µ –∫–Ω–∏–≥.
–î–≤–µ –±–∏–±–ª–∏–æ—Ç–µ–∫–∏ —Ä–∞–≤–Ω—ã, –µ—Å–ª–∏ –≤ –Ω–∏—Ö —Å–æ–¥–µ—Ä–∂–∞—Ç—Å—è –æ–¥–∏–Ω–∞–∫–æ–≤—ã–µ –Ω–∞–±–æ—Ä—ã –∫–Ω–∏–≥
(–∫–Ω–∏–≥–∏, –≤ —Å–≤–æ—é –æ—á–µ—Ä–µ–¥—å, —Å—á–∏—Ç–∞—é—Ç—Å—è –æ–¥–∏–Ω–∞–∫–æ–≤—ã–º–∏, –µ—Å–ª–∏ —É –Ω–∏—Ö –æ–¥–∏–Ω–∞–∫–æ–≤—ã–µ –Ω–∞–∑–≤–∞–Ω–∏—è –∏ –∞–≤—Ç–æ—Ä).
–ù–∞–ø—Ä–∏–º–µ—Ä:
```python
biblio1 = Library(name="–ë–∏–±–ª–∏–æ—Ç–µ–∫–∞ –ú–§–¢–ò")
biblio2 = Library(name="–õ–µ–Ω–∏–Ω–∫–∞")
book = Book(
    name="–í–µ–¥—å–º–∞",
    author="–§—Ä–∏—Ü –õ–µ–π–±–µ—Ä",
    year=1943
)

biblio1 = biblio1 + book
biblio2 = biblio2 + book

print(biblio1 > biblio2)   # False
print(biblio1 == biblio2)  # True, —Ç–∞–∫ –∫–∞–∫ 1 == 1

biblio1 += book

print(biblio1 > biblio2)   # True, —Ç–∞–∫ –∫–∞–∫ 2 > 1
print(biblio1 == biblio2)  # False
```

–£ –±–∏–±–ª–∏–æ—Ç–µ–∫–∏ –¥–æ–ª–∂–µ–Ω –±—ã—Ç—å "—á–∏—Ç–∞–µ–º—ã–π" –≤—ã–≤–æ–¥ –ø—Ä–∏ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–∏ `print`.
–ù–∞–ø—Ä–∏–º–µ—Ä:
```python
biblio = Library(name="–ë–∏–±–ª–∏–æ—Ç–µ–∫–∞ –ú–§–¢–ò")

    name="–î–≤–∞–∂–¥—ã —É–º–µ—Ä—à–∞—è (–ü–æ–≤–µ—Å—Ç—å –æ —Ç–æ–º, –∫–∞–∫ –ª—é–±—è—â–∞—è –ß–∂–æ—É –®—ç–Ω-—Å—è–Ω—å –ø—Ä–∏–≤–µ–ª–∞ –≤ —Å–º—è—Ç–µ–Ω–∏–µ –ª–∞–≤–∫—É –§–∞–Ω–µ–π)",
    author="-",
    year=1600
)

biblio = biblio + book

print(biblio)  # –ë–∏–±–ª–∏–æ—Ç–µ–∫–∞ –ú–§–¢–ò (—á–∏—Å–ª–æ –∫–Ω–∏–≥: 1)
```


## –ó–∞–¥–∞—á–∞ 2 (‚Ñö)

[–ò—Å—Ç–æ—á–Ω–∏–∫](https://github.com/Alvant/AdvancedPython/tree/master2022/labs/lab06#%D0%B7%D0%B0%D0%B4%D0%B0%D1%87%D0%B0-2-%E2%84%9A).

–ù—É–∂–Ω–æ –æ–ø—Ä–µ–¥–µ–ª–∏—Ç—å –∫–ª–∞—Å—Å *–†–∞—Ü–∏–æ–Ω–∞–ª—å–Ω–æ–≥–æ —á–∏—Å–ª–∞*: —á–∏—Å–ª–æ –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç—Å—è —á–∏—Å–ª–∏—Ç–µ–ª–µ–º (—Ü–µ–ª–æ–µ —Ü–∏—Å–ª–æ) –∏ –∑–Ω–∞–º–µ–Ω–∞—Ç–∞–ª–µ–º (–Ω–∞—Ç—É—Ä–∞–ª—å–Ω–æ–µ —á–∏—Å–ª–æ), –¥–æ–ª–∂–Ω—ã –±—ã—Ç—å –º–µ—Ç–æ–¥—ã –ø–æ —Å–ª–æ–∂–µ–Ω–∏—é, –≤—ã—á–∏—Ç–∞–Ω–∏—é, —É–º–Ω–æ–∂–µ–Ω–∏—é –∏ –¥–µ–ª–µ–Ω–∏—é —Ä–∞—Ü–∏–æ–Ω–∞–ª—å–Ω—ã—Ö —á–∏—Å–µ–ª.
–¢–∞–∫–∂–µ –¥–æ–ª–∂–µ–Ω –±—ã—Ç—å –º–µ—Ç–æ–¥ –ø–æ –ø–µ—Ä–µ–≤–æ–¥—É —Ä–∞—Ü–∏–æ–Ω–∞–ª—å–Ω–æ–≥–æ —á–∏—Å–ª–∞ –≤ –æ–±—ã—á–Ω–æ–µ `float` —á–∏—Å–ª–æ (–Ω–∞–ø—Ä–∏–º–µ—Ä –æ–±—ä–µ–∫—Ç, –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—é—â–∏–π —á–∏—Å–ª–æ `1/8` –¥–æ–ª–∂–µ–Ω –æ—Ç–æ–±—Ä–∞–∑–∏—Ç—å—Å—è –≤ `0.125`).

## –ó–∞–¥–∞—á–∞ 3 (¬´–¢–æ—á–∫–∏ –≤ –ø–æ–ª—è—Ä–Ω—ã—Ö –∫–æ–æ—Ä–¥–∏–Ω–∞—Ç–∞—Ö¬ª)

[–ò—Å—Ç–æ—á–Ω–∏–∫](https://github.com/Alvant/AdvancedPython/tree/master2022/labs/lab06#%D0%B7%D0%B0%D0%B4%D0%B0%D1%87%D0%B0-5-%D1%82%D0%BE%D1%87%D0%BA%D0%B8-%D0%B2-%D0%BF%D0%BE%D0%BB%D1%8F%D1%80%D0%BD%D1%8B%D1%85-%D0%BA%D0%BE%D0%BE%D1%80%D0%B4%D0%B8%D0%BD%D0%B0%D1%82%D0%B0%D1%85).

–ù–∞–¥–æ —Ä–µ–∞–ª–∏–∑–æ–≤–∞—Ç—å –∫–ª–∞—Å—Å `Point` –¥–ª—è –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏—è —Ç–æ—á–µ–∫ –≤ [–ø–æ–ª—è—Ä–Ω–æ–π —Å–∏—Å—Ç–µ–º–µ –∫–æ–æ—Ä–¥–∏–Ω–∞—Ç](https://en.wikipedia.org/wiki/Polar_coordinate_system) –Ω–∞ –ø–ª–æ—Å–∫–æ—Å—Ç–∏.
–¢–æ—á–∫–∞ –≤ –ø–æ—Ä—è—Ä–Ω–æ–π —Å–∏—Å—Ç–µ–º–µ –∫–æ–æ—Ä–¥–∏–Ω–∞—Ç –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç—Å—è —Ä–∞—Å—Å—Ç–æ—è–Ω–∏–µ–º –¥–æ –ø–æ–ª—é—Å–∞ *r* –∏ —É–≥–ª–æ–º *œÜ* –º–µ–∂–¥—É –Ω–∞–ø—Ä–∞–≤–ª–µ–Ω–∏–µ–º –Ω–∞ —Ç–æ—á–∫—É –∏–∑ –ø–æ–ª—é—Å–∞ –∏ –∑–∞—Ä–∞–Ω–µ–µ –≤—ã–±—Ä–∞–Ω–Ω—ã–º –Ω–∞–ø—Ä–∞–≤–ª–µ–Ω–∏–µ–º **l**.
–†–∞—Å—Å—Ç–æ—è–Ω–∏–µ –±—É–¥–µ–º —Å—á–∏—Ç–∞—Ç—å –±–µ–∑—Ä–∞–∑–º–µ—Ä–Ω—ã–º, –∞ —É–≥–æ–ª ‚Äî –≤—ã—Ä–∞–∂–µ–Ω–Ω—ã–º –≤ –≥—Ä–∞–¥—É—Å–∞—Ö.
–ù–∞–ø—Ä–∏–º–µ—Ä, —Å–æ–∑–¥–∞–Ω–∏–µ –æ–±—ä–µ–∫—Ç–∞, –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—é—â–µ–≥–æ —Ç–æ—á–∫—É, —É–¥–∞–ª—ë–Ω–Ω—É—é –æ—Ç –ø–æ–ª—é—Å–∞ –Ω–∞ —Ä–∞—Å—Å—Ç–æ—è–Ω–∏–µ 10 –ø–æ–¥ —É–≥–ª–æ–º –≤ 30 –≥—Ä–∞–¥—É—Å–æ–≤:
```python
p = Point(r=10, phi=30)
```

–î–æ–ª–∂–Ω–∞ –±—ã—Ç—å –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç—å —Å–∫–ª–∞–¥—ã–≤–∞—Ç—å —Ç–æ—á–∫–∏.
–ü—Ä–∏ —Å–ª–æ–∂–µ–Ω–∏–∏ –¥–≤—É—Ö —Ç–æ—á–µ–∫ —Å–∫–ª–∞–¥—ã–≤–∞—é—Ç—Å—è –∏—Ö —Ä–∞–¥–∏—É—Å—ã-–≤–µ–∫—Ç–æ—Ä—ã –ø–æ –ø—Ä–∞–≤–∏–ª—É —Å–ª–æ–∂–µ–Ω–∏—è –≤–µ–∫—Ç–æ—Ä–æ–≤.
–ò –∫–æ–æ—Ä–¥–∏–Ω–∞—Ç–∞–º–∏ —Ç–æ—á–∫–∏-—Ä–µ–∑—É–ª—å—Ç–∞—Ç–∞ –±—É–¥—É—Ç: –¥–ª–∏–Ω–∞ —Ä–µ–∑—É–ª—å—Ç–∏—Ä—É—é—â–µ–≥–æ –≤–µ–∫—Ç–æ—Ä–∞ –∏ —É–≥–æ–ª, –∫–æ—Ç–æ—Ä—ã–π –æ–Ω –æ–±—Ä–∞–∑—É–µ—Ç —Å –Ω–∞–ø—Ä–∞–≤–ª–µ–Ω–∏–µ–º **l**.
–ù–∞–ø—Ä–∏–º–µ—Ä:
```python
p1 = Point(r=1, phi=0)
p2 = Point(r=1, phi=90)
p3 = p1 + p2
# p3 ‚Äî —Ç–æ—á–∫–∞ —Å r = 1.41 (–ø—Ä–∏–º–µ—Ä–Ω–æ) –∏ phi = 45
```

–£ —Ç–æ—á–∫–∏ `Point` –¥–æ–ª–∂–Ω–æ –±—ã—Ç—å –ø–æ–Ω—è—Ç–Ω–æ–µ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏–µ –ø—Ä–∏ –≤—ã–≤–æ–¥–µ —Å –ø–æ–º–æ—â—å—é `print`.
–ù–∞–ø—Ä–∏–º–µ—Ä:
```python

print(p)  # (r = 1, œÜ = 10)
```

–¢–∞–∫–∂–µ –¥–æ–ª–∂–Ω–∞ –±—ã—Ç—å –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç—å –ø—Ä–æ–≤–µ—Ä–∏—Ç—å –¥–≤–µ —Ç–æ—á–∫–∏ –Ω–∞ —Ä–∞–≤–µ–Ω—Å—Ç–≤–æ.
–û–±—ä–µ–∫—Ç—ã –∫–ª–∞—Å—Å–∞ `Point` —Ç–æ—á–∫–∏ `p1` –∏ `p2` —Ä–∞–≤–Ω—ã, –µ—Å–ª–∏ –æ–Ω–∏ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—é—Ç –æ–¥–Ω—É –∏ —Ç—É –∂–µ —Ç–æ—á–∫—É –ø–ª–æ—Å–∫–æ—Å—Ç–∏:
```python

p2 = Point(1, 480)

print(p1 == p2)  # True
```


## –ó–∞–¥–∞—á–∞ 4 (¬´–ö–∏–Ω–æ¬ª)

[–ò—Å—Ç–æ—á–Ω–∏–∫](https://github.com/Alvant/AdvancedPython/tree/master2022/labs/lab05#%D0%B7%D0%B0%D0%B4%D0%B0%D1%87%D0%B0-1-%D0%BA%D0%B8%D0%BD%D0%BE).

–õ—é–¥–∏ –ª—é–±—è—Ç —Å–º–æ—Ç—Ä–µ—Ç—å –∫–∏–Ω–æ, –æ–¥–Ω–∞–∫–æ –¥–∞–Ω–Ω–æ–º—É —á–µ–ª–æ–≤–µ–∫—É —Ñ–∏–ª—å–º –º–æ–∂–µ—Ç –±—ã—Ç—å –Ω–µ —Ä–µ–∫–æ–º–µ–Ω–¥–æ–≤–∞–Ω –∫ –ø—Ä–æ—Å–º–æ—Ç—Ä—É –∏–∑-–∑–∞ –≤–æ–∑—Ä–∞—Å—Ç–Ω–æ–≥–æ –æ–≥—Ä–∞–Ω–∏—á–µ–Ω–∏—è...

–§–∏–ª—å–º (`Movie`) –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç—Å—è —Å–ª–µ–¥—É—é—â–∏–º–∏ –ø—Ä–∏–∑–Ω–∞–∫–∞–º–∏:
* –Ω–∞–∑–≤–∞–Ω–∏–µ (—Å—Ç—Ä–æ–∫–∞)
* —Ä–µ–∂–∏—Å—Å—ë—Ä (—Å—Ç—Ä–æ–∫–∞)
* –≥–æ–¥ –≤—ã–ø—É—Å–∫–∞ (—Ü–µ–ª–æ–µ —á–∏—Å–ª–æ)
* —Å—Ç—Ä–∞–Ω–∞ (—Å—Ç—Ä–æ–∫–∞)
* –ø—Ä–æ–¥–æ–ª–∂–∏—Ç–µ–ª—å–Ω–æ—Å—Ç—å –≤ –º–∏–Ω—É—Ç–∞—Ö (—Ü–µ–ª–æ–µ —á–∏—Å–ª–æ)
* –≤–æ–∑—Ä–∞—Å—Ç–Ω–æ–π —Ä–µ–π—Ç–∏–Ω–≥ ‚Äî –º–∏–Ω–∏–º–∞–ª—å–Ω—ã–π –¥–æ–ø—É—Å—Ç–∏–º—ã–π –≤–æ–∑—Ä–∞—Å—Ç –¥–ª—è –ø—Ä–æ—Å–º–æ—Ç—Ä–∞ (—Ü–µ–ª–æ–µ —á–∏—Å–ª–æ)

–£ —á–µ–ª–æ–≤–µ–∫–∞ –∂–µ (`Human`) –µ—Å—Ç—å —Å–ª–µ–¥—É—é—â–∏–µ –∞—Ç—Ä–∏–±—É—Ç—ã:
* –∏–º—è
* –ø–æ–ª
* –≥–æ–¥ —Ä–æ–∂–¥–µ–Ω–∏—è

–ß—Ç–æ–±—ã –ø–æ–Ω—è—Ç—å, –º–æ–∂–Ω–æ –ª–∏ –¥–∞–Ω–Ω–æ–º—É —á–µ–ª–æ–≤–µ–∫—É `human` —Å–º–æ—Ç—Ä–µ—Ç—å —Ñ–∏–ª—å–º `movie`,
—É –∫–∞–∂–¥–æ–≥–æ —Ñ–∏–ª—å–º–∞ –¥–æ–ª–∂–µ–Ω –±—ã—Ç—å –º–µ—Ç–æ–¥ `is_allowed(human)`,
–∫–æ—Ç–æ—Ä—ã–π –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç `True` –∏–ª–∏ `False` –≤ –∑–∞–≤–∏—Å–∏–º–æ—Å—Ç–∏ –æ—Ç —Ç–æ–≥–æ,
—Ä–∞–∑—Ä–µ—à–µ–Ω–æ –∏–ª–∏ –Ω–µ—Ç —á–µ–ª–æ–≤–µ–∫—É —Å–º–æ—Ç—Ä–µ—Ç—å —Ñ–∏–ª—å–º:
```python
movie = Movie(
  name="–≠—Ç–æ –±–µ–∑—É–º–Ω—ã–π, –±–µ–∑—É–º–Ω—ã–π, –±–µ–∑—É–º–Ω—ã–π, –±–µ–∑—É–º–Ω—ã–π –º–∏—Ä",
  director="–°—Ç—ç–Ω–ª–∏ –ö—Ä–∞–º–µ—Ä", year=1963,
  country="–°–®–ê", duration=210, age_rating=6

print(movie.is_allowed(human))  # True (—Å–µ–π—á–∞—Å 2023 –≥–æ–¥, –ø–æ—ç—Ç–æ–º—É –ù–µ–æ —Ç–æ—á–Ω–æ –º–æ–∂–Ω–æ —Å–º–æ—Ç—Ä–µ—Ç—å –ª—é–±—ã–µ —Ñ–∏–ª—å–º—ã)
```
